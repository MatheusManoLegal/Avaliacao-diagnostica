A estilização de páginas usando a semântica da progamação W3C refere-se a palicar estilos em elementos
HTML seguindo as diretrizes do W3C. Isso é feito principalmente usando CSS para controlar a aparência
dos elementos. Os seletores CSS são usados para direcionar os elementos qe se deseja estilizar, como seletores de tipo
classe,ID,descência e filho direto. Esses seletores permitem uma seleção precisa dos elementos HTML para aplicar estilos

As subclasses de CSS são usadas para alterar visualmente a página ou seus elementos, um exemplo é quando você passa o mouse por cima deste texto(hover)
